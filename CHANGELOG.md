# Changelog

## [Recent Updates] - Statistics Page Enhancement

### Changed

- **Recent Links Section**: Removed borders and replaced with vibrant solid gradient backgrounds
    - Each link card now has a unique color: Indigo, Purple, Pink, Blue, or Violet
    - Background gradients create depth with `from-[color]-500 to-[color]-600`
    - Text and icons are now white with semi-transparent backgrounds for better contrast
    - Stats badges use glassmorphism effect (white/20 with backdrop blur)

- **Performance Trend Chart**: Updated to use matching colors
    - Each data point and bar uses the same color scheme as the Recent Links
    - Individual colors for each link: Indigo → Purple → Pink → Blue → Violet
    - Consistent visual language across all statistics visualizations

### Visual Improvements

- **No Borders**: Clean, modern look without border clutter
- **Color Coded**: Each link is instantly distinguishable by its unique gradient
- **Better Contrast**: White text on vibrant backgrounds ensures readability
- **Glassmorphism**: Semi-transparent badges with backdrop blur for modern effect
- **Hover Effects**: Scale transformation on hover for interactivity

### Color Palette Used

1. **Indigo**: `from-indigo-500 to-indigo-600`
2. **Purple**: `from-purple-500 to-purple-600`
3. **Pink**: `from-pink-500 to-pink-600`
4. **Blue**: `from-blue-500 to-blue-600`
5. **Violet**: `from-violet-500 to-violet-600`
